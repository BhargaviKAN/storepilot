# StorePilot Project Plan

## Milestone 1 — Foundation
- Create GitHub repository
- Add README
- Add project folders
- Add synthetic dataset
- Make first commit

## Milestone 2 — Data preparation
- Inspect data types
- Check missing values and duplicates
- Validate ranges
- Create cleaned dataset

## Milestone 3 — KPI analysis
- Target achievement
- Conversion rate
- Average transaction value
- Sales per labour hour
- Accessory attachment rate

## Milestone 4 — Operations analysis
- Compare customer traffic with staffing
- Identify understaffed periods
- Identify overstaffed periods
- Analyse wait time and customer satisfaction

## Milestone 5 — Dashboard
- Build Streamlit app
- Add date, employee, and category filters
- Display KPIs and recommendations

## Milestone 6 — Final polish
- Add tests
- Add screenshots
- Improve documentation
- Write project findings
